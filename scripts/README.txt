Various scripts for creating the lists required by ToMaR.

Edit the files with values appropriate for your cluset setup.

Create a large test set of copied files from a small test set with createTestFiles.sh.  

Create the list of raw files with createListOfInputFilesRaw.sh

Use the other files to create the appropriate lists for ToMaR once the list of raw files has been created.

Launch Taverna using the tomarWorkflowLaunchertest execute in the taverna home directory(after editing to suit first).
